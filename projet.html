Le projet inclut un système d'authentification sécurisé ainsi qu'un ensemble complet de fonctionnalités CRUD (Create, Read, Update, Delete) autour d'un objet olympique spécifique (épreuves, athlètes, disciplines, etc.). APPLI OUF

Organiser son développement professionnel
Lien github

Réalisation, dans le cadre d’un travail de classe, de l’application Java SE "AppliOuf" pour l’entreprise fictive OUF. L’objectif était de développer une solution de gestion incluant : cartes prépayées, transactions, rendez-vous médicaux (côté utilisateur) ainsi que la gestion des produits et des stocks (côté administrateur). L’interface repose sur des boîtes de dialogue interactives via JOptionPane, organisées sous forme de menus. Les données sont stockées en mémoire à l’aide de structures List, sans persistance en base de données. Ce projet a permis de mettre en pratique les fondamentaux de la programmation orientée objet, la gestion de rôles, et l’organisation modulaire d’une application.

Mastering

Gérer le patrimoine informatique Travailler en mode projet Mettre à disposition des utilisateurs un service informatique Organiser son développement professionnel
Lien github

Mastering est une application web développée avec Spring Boot pour le back-end et Angular 19 pour le front-end. Elle permet de gérer efficacement l’organisation des soutenances de master, depuis la planification des sessions jusqu’à la génération des attestations. L'application facilite la coordination entre les étudiants, tuteurs, jurys et responsables en centralisant les informations, les documents et les communications liés à chaque soutenance.
