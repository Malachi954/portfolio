Ce projet a été réalisé dans le cadre d’une semaine de piscine sur le thème des Jeux Olympiques. L’objectif était de concevoir une application web permettant de recenser tous les participants français aux JO, tout en intégrant les fonctionnalités de base d’une application CRUD (Create, Read, Update, Delete).
